
1) Reflect on the differences between static and interactive visualizations.
Static visualizations can be fast and can be easily shared, while the interactive visualizations could allow users to drill down into the data and explore it in more detail.

2) Write a short paragraph discussing the advantages of using dashboards for real-time data exploration.
  Dashboards can provide real-time insights about key metrics and trends at a glance. They consolidate multiple insights into a single quick view and allow users to drill down into specifics. It also allows a central place for users to monitor key metrics.

3) Explain how interactive tools like Plotly and Dash can improve data communication in professional settings.
Interactive tools like Plotly and Dash allow users to create dynamic visualizations that can be manipulated in real-time. It allows users to filter data and explore insights that important to them. This allows for a self-serve model for reports usage.
